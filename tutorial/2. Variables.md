### Declare variables
```go
var str string = "Hello guys"  
fmt.Println("Hello world " + str)
```

- Short variable declaration
```go
numb := 1000  // only allowed inside a method
fmt.Println(numb)
```


### Rune
- Every single byte in a string is  a rune
- Go uses UTF 8
- Rune is a UTF 8 code point

### Pointer
